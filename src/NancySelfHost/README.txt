## INSTALL REDIS

https://github.com/MSOpenTech/redis/tree/2.8/bin/release
cmd
redis-server

OR

https://github.com/rgl/redis/downloads
cmd
net start redis

OR

Use Redis Vagrant from Scratchpad

## REDIS COMMANDS

LUSHDB       - Removes data from your connection's CURRENT database.
FLUSHALL     - Removes data from ALL databases.
